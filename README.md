# Target-Recognition-Console
基于Dlib的人脸识别和目标识别软件。  
[下载模型](https://github.com/Firemountaincold/Target-Recognition-Console/blob/main/module.rar)
## 方法：
用Aforge操作摄像头。    
用Dlib进行人脸识别和目标识别。
